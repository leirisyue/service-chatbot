# BƯỚC 2: Nếu chưa có Service Account:
# - Click "CREATE SERVICE ACCOUNT"
# - Name: gemini-chatbot-sa
# - Click "CREATE AND CONTINUE"
# - Select role: "Vertex AI User" 
# - Click "CONTINUE" → "DONE"

# BƯỚC 3: Tạo key
# - Click vào Service Account vừa tạo (hoặc đã có)
# - Tab "KEYS" 
# - "ADD KEY" → "Create new key"
# - Chọn "JSON" 
# - Click "CREATE"
# → File JSON sẽ được tải về

# BƯỚC 4: Copy toàn bộ nội dung file JSON đó
# và paste vào file service-account-key.json (tạo mới)
